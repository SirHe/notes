# 其他

老外比较喜欢先说事情，忽略各种条件。当把事情讲完之后，再一点一点地补充条件。比如：`Returns Get(V, P), with V converted to an Object with ToObject first if necessary.` **但我觉得条件更重要，有的事情如果条件不成立，根本就不会发生，也就是不存在**。所以理论上来说，中文的语序（语言顺序）更加合理。
